Web Interface
=============

A web interface for users to edit their information. Users login with their
LDAP credentials. Can be deployed using CherryPy's own server.

Dependencies:
* The ldapsearch utility must be installed on the server
* Python3.x:

    * CherryPy
    * Jinja 
    * py-postgresql
    * sqlalchemy
